# extract
tool to extract text from major document formats

## supported formats

- [x] pdf
- [x] msg
- [ ] eml (not implemented) 
- [x] ppt
- [ ] xls (not implemented)
- [ ] doc (not implemented)
- [x] rtf
- [x] html
- [x] pptx
- [x] xlsx
- [x] docx
